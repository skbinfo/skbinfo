
Research in **Bioinformatics, Genomics, AI/ML, Deep Learning, Big Data Analysis, and Plant Biotechnology**
Exploring:
* tRNA-derived ncRNAs (tncRNAs)
* Fusion transcripts
* Long non-coding RNAs
* Multi-omics data analysis
* AI-driven plant genomics research
Developing computational pipelines, databases, and web resources for plant genomics.
---
